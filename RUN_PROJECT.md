# How to Run the Project

## 🚀 Step-by-Step Commands

### Terminal 1: Start MongoDB (Run First!)
```powershell
cd C:\Users\Sathwik\Downloads\ExamShield
.\start-mongodb-27018.bat
```
**Keep this terminal open!**

---

### Terminal 2: Start Backend
```powershell
cd C:\Users\Sathwik\Downloads\ExamShield\backend-code
npm run dev
```
**Keep this terminal open!**

---

### Terminal 3: Start Frontend
```powershell
cd C:\Users\Sathwik\Downloads\ExamShield
npm run dev
```

---

### Open Website
👉 **http://localhost:8080**

---

## ✅ Order of Commands

1. **Terminal 1** → MongoDB (start-mongodb-27018.bat)
2. **Terminal 2** → Backend (npm run dev in backend-code)
3. **Terminal 3** → Frontend (npm run dev)
4. **Browser** → http://localhost:8080
